Assalamualaikum dan salam sejahtera kepada pembaca IIUM Confessions dan para admins. Terlebih dahulu, saya berterima kasih kepada pihak admins jika coretan ini diterbitkan. Walaupun tidak diterbitkan, terima kasih kerana sudi membaca.Saya mulakan dengan pengenalan diri saya. Nama saya Tia, anak sulung daripada tiga beradik. Sebagai anak sulung, saya sedar jika sesuatu berlaku kepada ibu bapa saya, secara automatik tanggungjawab keluarga dipikul oleh saya. Sebenarnya, saya merasakan bahawa saya mengalami kemurungan. Ya, saya pasti ramai pembaca akan menyuruh saya untuk berjumpa dengan doktor untuk kepastian. Ada beberapa sebab yang menyebabkan saya untuk tidak pergi berjumpa dengan doktor.Kisah saya bermula sewaktu saya sekolah rendah. Seorang pelajar yang pintar dan disayangi guru, kehidupan saya dipenuhi dengan momen ceria. Namun, semua itu berubah apabila saya mengalami epilepsi atau lebih dikenali sebagai sawan. Penyakit ini secara tidak langsung telah mengubah kehidupan saya. Penyakit saya dibicara oleh para guru tanpa segan-silu malah ada pelajar mendengarnya. Saya diejek dengan perkataan “sawan babi”, “wayar putus” dan beberapa perkataan yang sudah saya lupakan. Sampai satu ketika, saya pernah menangis kerana diejek sebegitu. Faktor ejekan tersebut mendorong saya untuk membuat keputusan memasuki sekolah menengah di mana tiada langsung pelajar sekolah rendah yang sama dengan saya.Alam sekolah menengah, saya menyembunyikan penyakit saya dan Alhamdulillah, saya tidak diserang sawan sehingga Tingkatan 3. Ketika itu, ayah saya bertugas di luar negara. Hanya ibu yang mengurus keluarga. Sewaktu penyakit ini menyerang, saya melihat betapa lemahnya seorang ibu. Dia yang tak pernah tunjuk penatnya, menangis melihat saya diserang sawan. Malah ayah saya turut dihubungi dan diminta pulang dengan kadar segera. Pada ketika itu, saya tidak merasakan apa-apa. Tumpuan saya hanya pada PMR. Saya lulus dengan cemerlang and juga bersyukur kerana rakan-rakan serta para guru langsung tidak pernah menyebut soal penyakit saya. Namun begitu, kami sekeluarga terpaksa berpindah atas tuntutan kerja ayah.Di sini, bermula episod kemurungan saya. Saya yang berasal daripada sekolah perempuan sahaja memasuki sekolah di mana pelajarnya bercampur lelaki dan perempuan. Membawa nama sekolah perempuan saya pada ketika itu telah menyebabkan saya dipulau tanpa ditanya apa-apa. Saya hanya berkawan dengan rakan-rakan cina. Pada ketika ini, saya merasakan kehidupan saya sunyi dan saya memberontak secara diam terhadap ibu bapa saya. Pelajaran saya mulai merosot dan kedatangan saya ke sekolah diselangi dengan ponteng di bawah pengawasan ibu saya. Saya menjadi seorang yang lebih gemar bersendirian dan mula mencipta rakan imaginasi. Sehingga sekarang ibu bapa saya tidak tahu soal rakan imaginasi saya. Kejadian pulau itu tidak berlarutan lama. Setelah saya menjelaskan nama sekolah lama dan berbuat baik, ada yang sudi berkawan dengan saya. Namun begitu, saya tidak meletakkan sepenuh kepercayaan terhadap mereka. Malah, pada waktu itu saya tidak mempercayai sesiapa kecuali diri saya.Tamat persekolahan, saya banyak menghabiskan masa di sosial media. Medium ini menjadi tempat saya meluahkan perasaan malah rakan maya saya banyak membantu saya. Saya meneruskan pengajian ke sebuah universiti dan di sana saya mula mempercayai rakan sekelas saya. Ibarat kaca yang telah dicantum berderai semula, begitu juga kepercayaan saya. Saya ditinggalkan oleh rakan sekelas saya malah dipulau secara tidak langsung oleh mereka. Bahkan, para pensyarah dapat melihat apa yang dilakukan oleh mereka terhadap saya. Saya mula bergerak secara solo dan perangai ini saya bawa ke pengajian ijazah. Sewaktu pengajian ijazah, saya bertemu dengan rakan-rakan yang sekarang ini saya anggap keluarga kedua saya.  Soal percaya, ya saya masih tidak mampu mempercayai manusia sepenuhnya. Bahkan, diri saya sendiri saya tak mampu untuk percaya. Saya mengalami kemurungan kali ketiga sewaktu tahun ketiga pengajian ijazah. Saya kehilangan seorang rakan baik dan pada waktu yang sama saya mengetahui bahawa orang yang saya menaruh harapan hanyalah mempermainkan saya. Kehilangan rakan baik ditambah pula kesibukan membuat tesis telah mejadikan saya terlampau stres. Apabila stres saya memuncak, saya mula memikirkan tentang kesedihan saya dan tidak melakukan sebarang kerja. Hanya tidur dan menyelubungi diri saya dengan selimut. Air mata gugur berkali-kali. Tiada siapa pun tahu. Malah sehingga sekarang, apabila kemurungan melanda, saya hanya mampu menangis dalam senyap tanpa pengetahuan sesiapa.Saya tidak pasti berapa ramai lagi pembaca yang sedang membaca kisah saya. Berbalik kepada soalan mengapa saya tidak berjumpa dengan doktor untuk kepastian. Saya yang mempunyai tahap keyakinan diri yang lemah. Ibu saya menyedari hal ini dan sering membezakan saya dengan adik saya. Hal ini telah mengakibatkan diri saya berada dalam keadaan stres. Selain itu, saya merupakan seorang yang pendiam dan hanya berbicara jika perlu. Berat mulut, itulah ungkapan ibu saya kepada saya. Ya, saya mampu untuk pergi ke hospital sendiri. Namun, kehidupan saya sentiasa dibayangi ibu bapa saya. Tidak kah mereka pelik untuk apa saya pergi ke klinik atau hospital sedangkan saya sihat? Malah setiap pergerakan keluar masuk rumah saya pasti ditemani oleh ibu bapa saya. Memberitahu soal kemurungan kepada ibu bapa? Kepercayaan saya terhadap mereka belum pulih sepenuhnya. Saya juga masih ingat sesi di hospital di mana ayah saya berpura-pura menghidap sawan sedangkan saya yang sakit. Wajarkah saya untuk membebani hati serta kepala mereka dengan kemurungan saya?Jika dahulu, saya hanya menangis. Tapi kini, saya membayangkan pisau dan saya pasti anda memahami apa yang saya bayangkan. Saya sedar kemurungan saya semakin teruk dan saya perlu bantuan segera. Saya berharap agar saya cepat mendapatkan pekerjaan dan keluar daripada rumah. Dengan cara ini, saya dapat pergi mendapatkan bantuan tanpa pengetahuan ibu bapa.Saya berharap sangat para pembaca dapat membantu saya secara tidak langsung.
